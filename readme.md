## To run this project

1. Clone it

```bash
  git clone https://github.com/saqib40/Shell
```

2. Move to src directory

```bash
  cd src
```

3. Build the executable

```bash
  make
```

4. Run the executable

```bash
  ./main
```

5. Clean the executable

```bash
  make clean
```
